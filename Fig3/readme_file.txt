% time series;  
%--------------------------------------------------------------------------

% of the NGRC system from the given experimental time series; afterwards, the

% is anticipating the experimental time series; the time of anticipation is
% varied: [tau, 2*tau, ..., n*tau];
%--------------------------------------------------------------------------
