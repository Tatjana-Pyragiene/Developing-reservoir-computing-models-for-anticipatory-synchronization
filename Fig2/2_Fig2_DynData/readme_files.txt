% In this directory we compute the dynamics for the case of Lorenz system, 
% for graph Fig1(c);
%----------------------------------------------------------------------------------
% Main_Lorenz_Y_dyn.m -> this is the main program in this directory;
% This function uses the following auxillary functions:
%----------------------------------------------------------------------------------
% ReadParametersLS.m -> this function opens the data files and reads from them
% all necessary parameters for the case of the Lorenz system; this function
% is used in the main file; 
%----------------------------------------------------------------------------------
% Read_lam_icond.m -> this function opens the data file and reads the data
% from array lamIcond, containing the optimal CLE and the number of initial 
% conditions;
%----------------------------------------------------------------------------------
% DynFiles.m -> this function first computes the time series from the Lorenz system;
% next it performs the numerical integration of the NGRC system [Eq.(38)], which
% anticipates the time series of Lorenz system (its x(t) variable); this function
% is used in the main file;
%----------------------------------------------------------------------------------
% predict.m -> this function is used by function [DynFiles.m]; it performs the numerical 
% integration of the NGRC system [Eq.(38)], which anticipates the time series 
% of Lorenz system ( its x(t) variable); in output it gives the dynamics of
% NGRC and original Lorenz system (x(t));
%----------------------------------------------------------------------------------
% WriteFiles.m -> this function records in the data files the results of 
% computations by program [DynFiles.m]; 
%----------------------------------------------------------------------------------
% cell_mas.m -> this funtions is used by function [WriteFiles.m] when
% composing structures of data; these structures will be then recorded in the 
% data *.mat files;
%--------------------------------------------------------------------------
% Lorenz_signal.m -> this function computes the time series of the Lorenz
% system; this function is used by the function [DynFiles.m]
%--------------------------------------------------------------------------
% The file [predict.m] uses auxiliary functions for the NGRC algorithm:
%----------------------------------------------------------------------------
% generate_all_exp_matrices.m
% generateCheb_feat_vect.m
% generateChebyshevs.m
% generateChebyshevsDiffLyap.m
% genExponents.m
% matr.mat -> this data file contains the matrix of powers for Chebyshev polynomials;
% this file is generated by function [generate_all_exp_matrices.m];
