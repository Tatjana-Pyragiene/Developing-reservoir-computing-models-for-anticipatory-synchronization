% In this directory we compute the dynamics for the case of Roessler system, 
% for graphs Fig.1(c), Fig.1(d);
%----------------------------------------------------------------------------------
% Main_Rossler_Y_dyn.m -> this is the main program in this directory;
% This function uses the following auxiliary functions:
%---------------------------------------------------------------------------------
% ReadParametersRS.m -> this function opens the data files and reads from them
% all necessary parameters for the case of the Roessler system; 
%----------------------------------------------------------------------------------
% Read_lam_icond.m -> this function opens the data file and reads the data
% from array lamIcond, containing the optimal CLE and the number of initial 
% conditions;
%----------------------------------------------------------------------------------
% DynFiles.m -> this function first computes the time series from the Roessler system;
% next it performs the numerical integration of the NGRC system [Eq.(38)], which
% anticipates the time series of Roessler system (its y(t) variable);
%----------------------------------------------------------------------------------
% predict.m -> this function is used by function [DynFiles.m]; it performs the numerical 
% integration of the NGRC system [Eq.(38)], which anticipates the time series 
% of Roessler system (its y(t) variable); in output, it gives the dynamics of
% NGRC and original Roessler system (y(t));
%----------------------------------------------------------------------------------
% WriteFiles.m -> this function records in the data files the results of 
% computations by program [DynFiles.m]; this function is used in the main file; 
%----------------------------------------------------------------------------------
% cell_mas.m -> this funtions is used by function [WriteFiles.m] when
% composing structures of data; these structures will be then recorded in the 
% data *.mat files;
%--------------------------------------------------------------------------
% Roessler_signal.m -> this function computes the time series of the Roessler
% system; this function is used by the function [DynFiles.m]
%--------------------------------------------------------------------------
% The file [predict.m] uses auxiliary functions for the NGRC algorithm:
%----------------------------------------------------------------------------

% generateChebyshevsDiffLyap.m
